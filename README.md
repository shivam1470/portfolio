# Shivam Mishra | 3D Portfolio

A high-performance, interactive 3D portfolio built with **React**, **TypeScript**, **Three.js**, and **Tailwind CSS**. This project showcases my journey as a Senior Software Engineer through immersive storytelling and modern web technologies.

## 🚀 Live Demo
- **Portfolio:** https://shivam-mishra-sde.netlify.app/
- **Resume:** Included in the `/public` directory.

## ✨ Key Features
- **Interactive 3D Scenes:** Stylized procedural models (Laptop, Skill Cloud) built with `React Three Fiber`.
- **Dynamic Skill Cloud:** A floating, color-coded 3D sphere showcasing 30+ technical skills.
- **Work Experience Timeline:** A responsive vertical timeline of my professional journey at Deel, Wakencode, and more.
- **Projects Showcase:** Real-world projects including Dairy Farm Management and Space Science Explorer.
- **Medium Integration:** A dedicated section for technical articles fetched directly from my Medium profile.
- **Modern UI/UX:** Fully responsive design with dark mode, smooth scrolling, and `Framer Motion` animations.

## 🛠️ Tech Stack
- **Frontend:** React 19, TypeScript, Tailwind CSS
- **3D Engine:** Three.js, React Three Fiber (R3F), @react-three/drei
- **Animations:** Framer Motion
- **Build Tool:** Vite

## 📦 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shivam1470/portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
